# Infrastructure

Runtime/deployment configuration and supporting infrastructure assets.

- `nginx/`: Load balancer config (routing, health checks, websocket proxying).
- `redis/`: Redis config and helper scripts (Pub/Sub channels, local config, observability).
- `supabase/`: Database schema notes, migrations, SQL, and Supabase project setup docs.
- `docker/`: Docker helper files (compose overrides, local dev tooling, build assets).
